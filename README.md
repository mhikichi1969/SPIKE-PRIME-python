# SPIKE-Prime-python
SPIKE Prime でpythonコードを動作させる方法について

いくつか考えられる方法についてのメモ
## 純正ソフトウェアの利用
プロジェクトをPython形式で作成することが可能。

単一ファイルでの開発のみに対応

ダウンロードモードのみに対応、ストリーミングモードはScrachベースの環境のみ動作する

## ローカルで作成したPythonファイルをダウンロードモードで利用
大まかには以下の手順

-スロット領域の確保

-ファイルの転送

-実行
